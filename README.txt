BloodPressure PWA 完整版

部署到 GitHub Pages：
1. 將此 ZIP 解壓縮。
2. 把資料夾內的 index.html、styles.css、app.js、sw.js、manifest.webmanifest 以及 icons 資料夾，上傳到 GitHub Repository 根目錄。
3. GitHub Repository → Settings → Pages。
4. Source 選 Deploy from a branch。
5. Branch 選 main，Folder 選 /(root)，按 Save。
6. 等待網站顯示 Your site is live。
7. 用 iPhone Safari 開啟 GitHub Pages 網址。
8. 點 Safari 分享 → 加入主畫面。

功能：
- 自動記錄日期與時間
- 收縮壓、舒張壓、心跳、備註
- 統計最近值、平均值、本月筆數
- 趨勢圖
- 搜尋與排序
- 編輯與刪除紀錄
- CSV 匯入、匯出與分享
- 深色模式
- 離線使用
- 血壓提醒

重要：
資料主要存在目前裝置的瀏覽器本機儲存空間。
請定期使用「分享備份」或「匯出 CSV」，再選「儲存到檔案」→ iCloud Drive。
