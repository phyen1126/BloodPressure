BloodPressure PWA

安裝：
1. 將本資料夾部署到 HTTPS 網站空間，例如 GitHub Pages、Cloudflare Pages 或 Netlify。
2. 用 iPhone Safari 開啟網站。
3. 點 Safari 分享按鈕 → 加入主畫面。

使用：
- 資料會先保存在 iPhone 瀏覽器的本機儲存空間。
- 請定期點「分享備份」或「匯出 CSV」。
- 在分享表單選「儲存到檔案」→ iCloud Drive，即可備份。
- CSV 可用 iPhone Numbers、Microsoft Excel、Windows 或 Mac Excel 開啟。

限制：
網頁無法在未經使用者確認的情況下，持續直接改寫指定的 iCloud Drive 檔案。
清除 Safari 網站資料可能刪除本機紀錄，因此務必定期匯出備份。
